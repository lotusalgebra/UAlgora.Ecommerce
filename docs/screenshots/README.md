## Screenshot Guidelines

1. **Resolution**: Capture at 1920x1080 or higher
2. **Format**: Save as PNG
3. **Annotations**: Avoid adding annotations in screenshots
4. **Data**: Use realistic demo data (not "test" or "lorem ipsum")
5. **Browser**: Use Chrome in incognito mode (clean UI)
6. **Theme**: Use light mode for consistency

## Capturing Screenshots

### Windows
- Press `Win + Shift + S` for Snipping Tool
- Or use ShareX for advanced capture

### Mac
- Press `Cmd + Shift + 4` for selection
- Press `Cmd + Shift + 5` for more options

### Browser DevTools
- Press F12 to open DevTools
- Click Device Toolbar for responsive views
- Use "Capture full size screenshot" in more options
